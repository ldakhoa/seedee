# Introduction

`seedee` is an open-source iOS CI/CD toolkit. While many projects use Fastlane, this toolkit provides a convenient way to write configuration files; that is, `seedee` config can use with Swift, the most familiar and loved language in the iOS world

## Features
- Easy to setup
- Config file can write in Swift
- Integrates with all major CI systems

## Major dependencies
- [Swift Argument Parser](https://github.com/apple/swift-argument-parser) for CLI
- [Glob](https://github.com/Bouke/Glob) for Glob files
- [ColorizeSwift](https://github.com/mtynior/ColorizeSwift) for terminal log styling
- [Swift Collections](https://github.com/apple/swift-collections) for additional data structures
